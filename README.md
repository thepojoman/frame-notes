# Frame Notes

A small film photography field notebook for exposure recommendations and frame logging.

## Publish With GitHub Pages

1. Create a new public GitHub repository named `frame-notes`.
2. Upload these files to the repository:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `manifest.webmanifest`
   - `service-worker.js`
   - `icon.svg`
3. In the GitHub repository, open **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
6. Set **Branch** to `main` and folder to `/root`.
7. Click **Save**.
8. After GitHub finishes deploying, open the Pages URL on your iPhone in Safari.
9. Tap Share, then **Add to Home Screen**.

Your site will usually be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/frame-notes/
```

## Notes

- Location access works only from secure origins such as GitHub Pages HTTPS.
- Saved frames live in the browser's local storage on each device.
- Use **Export CSV** to keep a backup of your frame log.
